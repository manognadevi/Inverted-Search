# Inverted Search System

## Overview
The Inverted Search System is a C-based application designed to perform fast and efficient keyword searching across multiple text files using indexing techniques and data structures.

This project creates an inverted index that maps words to the files in which they appear, enabling optimized search operations similar to search engine functionality.

## Features
- Efficient keyword-based searching
- Inverted indexing implementation
- Hash table data structure
- Linked list traversal
- File handling operations
- Dynamic database creation and updating
- Fast retrieval of search results

## Technologies Used
- C Programming
- Data Structures
- File Handling
- Hashing Techniques
- Linked Lists

## Applications
- Search Engines
- Text Retrieval Systems
- Document Indexing
- Information Retrieval Systems
